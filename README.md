# TaskManagementApi

API REST desarrollada en ASP.NET Core para la gestión de tareas (CRUD).

## Descripción
Este proyecto permite crear, consultar, actualizar, completar y eliminar tareas
mediante una API REST probada con Postman.

## Tecnologías utilizadas
- ASP.NET Core
- C#
- Visual Studio 2022
- Postman
- Git y GitHub

## Endpoints principales

GET /api/tareas  
POST /api/tareas  
PUT /api/tareas/{id}  
PATCH /api/tareas/{id}/completar  
DELETE /api/tareas/{id}

## Ejecución del proyecto
1. Abrir el proyecto en Visual Studio
2. Ejecutar con IIS Express o Kestrel
3. Probar endpoints en Postman
